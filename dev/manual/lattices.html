Lattices.build â€‹

Lattices.build is used to build a Lattices.Lattice instance for common lattice structures, such as a hypercube.

For instance, to build a simple chain lattice of length 7 with the edge length 2.0 and periodic boundary conditions, run

julia
Lattices.build(
    :Hypercube, [7,], 2.0;
    periodic=[true,]
)
NeuralQuantumStates.Lattices.Lattice{Int64, Float64, 1, 1}

For a cuboid lattice of shape [3,4,5] with the edge length 2.0 and periodic boundary conditions in the second dimension, run

julia
Lattices.build(
    :Hypercube, [3, 4, 5], 2.0;
    periodic=[false, true, false]
)

Lattices.vertices â€‹

Lattices.vertices is used to get the labels and the label data from of all the vertices of a given a Lattices.Lattice instance:

julia
lat = Lattices.build(
    :Hypercube, [3, 4, 5], 2.0;
    periodic=[false, true, false]
);
labels, label_data = Lattices.vertices(lat.shape, lat.basis);
